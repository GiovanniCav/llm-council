# GLOBAL DOCTRINE: CORE PRINCIPLES

## 1. PARA+GTD Hybrid
- **00_Inbox**: Triage zone.
- **01_Projects**: Active initiatives with deadlines.
- **02_Areas**: Ongoing responsibilities.
- **03_Resources**: Static knowledge & reference assets.
- **04_Archive**: Completed/Inactive items.

## 2. Communication Rules
- Use **Clinical Density** (Strategic Partner tone).
- Avoid GPT fluff; prioritize actionable friction mapping.
- **ZERO URL HALLUCINATION**: Never guess or "make up" a URL. If the exact link is unknown, use an all-caps placeholder (e.g., `[INSERT LINK TO ASSET]`).

## 3. Data Integrity
- **Manifest-First**: Update `MANIFEST.jsonl` before file operations.
- **Local Primacy**: Local markdown/JSON is the SSOT.
