# LLM Model Leaderboards & Pricing Resources

## Instructions for Meta-Assessment Phase
When evaluating Roundtable performance, the Meta-Assessment MUST:
1. **Reference the models used** in each phase and compare against the current leaderboard rankings below.
2. **Flag model staleness:** If the models in `ROUNDTABLE_MODELS` config have dropped in ranking since last check, recommend specific replacements with names and pricing.
3. **Check for newcomers:** Periodically scrape Artificial Analysis (primary link below) for models that have entered the top 20 since the last check. If a new model offers better quality/price ratio for a specific phase, flag it.
4. **Include in every META-ASSESSMENT output:** A one-line "Model Currency" status, e.g.: _"Models current as of 2026-02-23. Gemini 3.1 Pro remains #1. No action needed."_ or _"⚠️ New model X has overtaken Y for the Sieve phase. Consider updating RT_MODEL_SIEVE."_

Reference links for the Roundtable Engine's model selection and Meta-Assessment phase.
Check these periodically to update the `ROUNDTABLE_MODELS` config.

## Primary (Live Leaderboards)

1. **[Artificial Analysis](https://artificialanalysis.ai/leaderboards/models)**
   Ranks 100+ models across intelligence, price, speed, latency, and context window.
   Uses blended price metric (3:1 input/output ratio). **Best single bookmark.**

2. **[PricePerToken.com/trends](https://pricepertoken.com/trends)**
   Benchmark scores vs. pricing over time. Has an MCP server for real-time pricing lookups.

3. **[LLM-Stats.com](https://llm-stats.com/ai-trends)**
   Trend data showing pricing trajectories and benchmark progressions over time.
